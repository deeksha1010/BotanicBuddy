# 🌿 PlantCare Web App
PlantCare is a React-based front-end web application designed to help users manage and care for their plants. It provides useful features like watering reminders, plant health tips, and an interactive plant catalog.

## Project Overview
The PlantCare app allows users to:
🌱 Track plant care schedules: Set reminders for watering, fertilizing, and more.
📝 Maintain a plant log: Add and track multiple plants with customized care schedules.
📚 Explore plant care tips: Get plant-specific advice and health tips for a thriving garden.
🔍 Search plant catalog: Explore a curated list of popular houseplants with details on care needs.

This project is built using React and provides an intuitive user interface to ensure that plant care becomes easier for everyone!
